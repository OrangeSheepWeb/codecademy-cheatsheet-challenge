Background elements properties

Property/Attribute Value Description
background-color Color on Hex format #FFFFF; or rgb(rrr,ggg,bbb); value Background color for the element selected
background-image url('../relative path to the image'); Set an image as background of an element
background-repeat repeat-x, repeat-y, no-repeat Specifies how background images are tiled after they have been sized and positioned.

Margin, Padding & Border

Property/Attribute Value Description
margin 4 values in px. Top, Right, Bottom, Left (Clockwise) Shorthand property to set values for the thickness of the margin area. If left is omitted, it is the same as right. If bottom is omitted it is the same as top, if right is omitted it is the same as top. Negative values for margin properties are allowed, but there may be implementation-specific limits.
padding 4 values in px. Top, Right, Bottom, Left (Clockwise) Shorthand property to set values for the thickness of the padding area. If left is omitted, it is the same as right. If bottom is omitted it is the same as top, if right is omitted it is the same as top. The value may not be negative.
border Width, style and color Shorthand property for setting border width, style, and color.
